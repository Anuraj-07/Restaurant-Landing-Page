# 🍽️ Savoria Restaurant Landing Page

A modern, fully responsive restaurant landing page built using HTML, CSS and JavaScript.

## 🌐 Live Demo

https://anuraj-07.github.io/Restaurant-Landing-Page/

---

## ✨ Features

- Responsive Design
- Dark / Light Mode
- Sticky Navigation
- Active Navigation Highlight
- Smooth Scrolling
- Scroll Progress Bar
- Loading Screen
- Animated Counters
- Chef's Special Section
- Reservation Form
- Customer Testimonials
- Food Gallery
- Ripple Button Animation
- Back To Top Button
- SEO Optimized
- Lazy Loading Images

---

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (ES6)
